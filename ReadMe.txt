Proyecto Fractal  
Este es creado para el manejo de  ordernes
Esta devidido en 2 proyecto:
    El back.- Generado en Spring Boot , que sirve como servidor de la información, y que guarda los datos en una base datos creada en mongo db
    El frond.-Generado con el frameword react para el manenjo de la información y peticiones, y para el manejo de los estilos tenemos al frameword bootstrap.
